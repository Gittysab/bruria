# גלריית תמונות קבועות

תמונות שיושבות כאן מוצגות במצגת לכל מי שנכנס לאתר (מכל מחשב/טלפון).

## איך מוסיפים תמונות
1. שמרו את קובצי התמונה בתיקייה הזו (למשל `photo1.jpg`, `photo2.jpg`).
2. הוסיפו את שמות הקבצים לרשימה ב‑`gallery.json`:

```json
{
  "images": ["photo1.jpg", "photo2.jpg"],
  "music": "song.mp3"
}
```

- `images` — רשימת שמות קובצי התמונות (לפי הסדר שיוצגו).
- `music` — שם קובץ מוזיקה לרקע (לא חובה; אפשר `null`).

3. דחפו את השינויים ל‑GitHub והאתר יתעדכן אוטומטית תוך כדקה.
